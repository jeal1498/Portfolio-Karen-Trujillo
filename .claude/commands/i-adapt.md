Follow the impeccable skill workflow for the `adapt` sub-command.

## Setup (required before proceeding)

1. Run `node .claude/skills/impeccable/scripts/context.mjs` to load project context. If you've already run it this session, skip.
2. Read `.claude/skills/impeccable/reference/adapt.md` — this defines the full flow. Follow it exactly.
3. Read at least one representative component/page file and the CSS/token system to understand the current design.
4. Determine the register: if this is a landing page / portfolio / marketing site, read `.claude/skills/impeccable/reference/brand.md`. If it is app UI / dashboard / tool, read `.claude/skills/impeccable/reference/product.md`.

Then execute the `adapt` command as defined in `reference/adapt.md`.
