Follow the impeccable skill. This is the main entry point — it handles design, redesign, shape, critique, audit, polish, clarify, distill, harden, optimize, adapt, animate, colorize, extract, and any other frontend interface improvement.

## Setup (required before proceeding)

1. Run `node .claude/skills/impeccable/scripts/context.mjs` to load project context. If you've already run it this session, skip.
2. Read `.claude/skills/impeccable/SKILL.md` for the full skill definition.
3. If the user invoked a sub-command (craft, shape, audit, polish, etc.), read `.claude/skills/impeccable/reference/<command>.md` next.
4. Read at least one representative component/page file and the CSS/token system.
5. Determine the register: landing page/portfolio/marketing → read `reference/brand.md`. App UI/dashboard/tool → read `reference/product.md`.

Then proceed as directed by the skill and any sub-command reference.
