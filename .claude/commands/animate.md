Follow the impeccable skill workflow for the `animate` sub-command.

## Setup (required before proceeding)

1. Run `node .agents/skills/impeccable/scripts/context.mjs` to load project context. If you've already run it this session, skip.
2. Read `.agents/skills/impeccable/reference/animate.md` — this defines the full flow. Follow it exactly.
3. Read at least one representative component/page file and the CSS/token system to understand the current design.
4. Determine the register: if this is a landing page / portfolio / marketing site, read `.agents/skills/impeccable/reference/brand.md`. If it is app UI / dashboard / tool, read `.agents/skills/impeccable/reference/product.md`.

Then execute the `animate` command as defined in `reference/animate.md`.
